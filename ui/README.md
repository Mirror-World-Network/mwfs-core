# CDWH Client UI #

## Compile and run in npm
0. run `npm i`
1. run `npm run generate_theme`
2. run `npm run dev`
3. access URL: [http://localhost:4000](http://localhost:4000)

## Build and run in Client
1. run `npm run build`
2. startup CDWH service
3. access URL: [http://localhost:8215](http://localhost:8215)
